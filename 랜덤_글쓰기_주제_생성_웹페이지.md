# 랜덤 글쓰기 주제 생성 웹페이지

## 개요

학생들에게 적합한 글쓰기 주제를 생성하고 뽑아주는 웹페이지

---

## 목표 및 대상

초등학교 학생(저,중,고학년)

---

## 핵심 기능

- 학생 연령에 맞는 주제 생성기능
  - 생성되는 주제의 수를 선택하는 기능
- 생성된 주제 중 뽑아서 보여주는 기능
  - 생성된 주제 중에서 뽑을 수 있는 주제의 수를 선택하는 기능
- 교사가 마음대로 몰래 주제를 뽑을 수 있도록 하는 기능

---

## 화면 구성

주제를 보여줄 때 슬롯머신 처럼 재밌게 보여주는 화면

---

## 기술 스택

- 주제가 생성되었을 때 효과음 나오는 기능
- 생성된 주제 중에서 글쓰기 주제가 뽑혔을 때 팝업으로 보여주고 효과음 나오는 기능
- 프론트엔드
  - react + vite
